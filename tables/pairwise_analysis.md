## 6. 核心发现

### 三个最强区分维度

1. **FORCE（力量）**
   - 区分轻抛（diu）vs 重摔（shuai摔）
   - 力量等级：diu < reng < pao < tou < shuai甩 < shuai摔

2. **HAND（手高度）**
   - 区分低抛（diu）vs 高投（tou）
   - 高度等级：diu < reng < shuai甩 < pao < shuai摔 < tou

3. **VD（垂直方向）**
   - 区分向上抛（大部分）vs 向下摔（shuai摔, schmettern, pfeffern）

### 特殊区分模式

- **ARM（手臂姿势）**：区分直臂（diu, toss）vs 弯臂（tou, throw, hurl）
- **HD（水平方向）**：区分向前（大部分）vs 向侧（shuai甩）

### 动词聚类

**力量型**（FORCE高）：shuai摔, hurl, schmettern, pfeffern
**高度型**（HAND高）：tou, throw
**轻柔型**（FORCE低, HAND低）：diu, toss
**侧向型**（HD=sidewise）：shuai甩
