想象你处于一个空旷房间中。

你的面前有一张桌子。

桌面上放置着一个手掌大小的物体。

目标动作：

{verb}

请采用你认为最自然、最典型的方式执行该动作。

若存在多种可能的执行方式，请选择最典型的一种。

请根据以下动作编码标准，对该动作进行编码。

动作标准编码如下：

请按照FORCE,ARM,HAND,VD,HD五个维度输出编码

FORCE
执行动作时施加于物体的力量大小

5 = 非常强
4 = 强
3 = 中等
2 = 弱
1 = 非常弱

ARM
动作开始前手臂的初始状态

1 = 手臂伸直
0 = 手臂弯曲

HAND
动作开始前手部停留的初始高度

0 = 地面高度

1–9 = 地面高度与执行者身高之间的相对高度

10 = 执行者身高

11 = 高于执行者身高一个单位

12 = 高于执行者身高两个单位

VD
执行动作时手部的主要垂直运动方向

1 = 向下
0 = 向上

HD
执行动作时手部的主要水平运动方向

1 = 向前
0 = 向侧方

若动作包含多个运动阶段，请以最主要的运动阶段进行判断。

所有变量必须输出整数编码值。

仅输出一个JSON对象。

格式如下：

{
  "FORCE": 0,
  "HAND": 0,
  "ARM": 0,
  "VD": 0,
  "HD": 0
}

除JSON对象外不要输出任何其他内容。
